Generic: node-children
==============================

Returns the child elements of an #<xml-element>#.


Exported from:
------------------------------
- #Module simple-xml#


Modifiers:
------------------------------
sealed


Arguments:
------------------------------
element - An instance of #<xml-element>#. 


Values:
------------------------------
children - An instance of #<stretchy-object-vector># containing instances of
           #<xml-element>#.
