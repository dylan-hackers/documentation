Function: xpair
==============================

Puts an element at the head of a list. Occasionally useful as a value to be
passed to a fold or other higher-order procedure.


Exported from:
------------------------------
- #Module sequence-utilities#


Arguments:
------------------------------
list - An instance of #<list>#. 
elt  - An instance of #<object>#. The element to add to the front of the list.


Values:
------------------------------
new-list - An instance of #<list>#.
