Function: always
==============================

Returns a function that always returns a particular object. The function can
be called with any number of arguments. The function ignores its arguments and
always returns _object_.


Exported from:
------------------------------
- #Module common-dylan#
- #Module Dylan#


Arguments:
------------------------------
object - An instance of #<object>#. 


Values:
------------------------------
function - An instance of #<function>#.
