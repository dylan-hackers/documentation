Function: check-type
==============================

Checks an object to ensure that it is an instance of a specified type. If it
is not, signal an error of type #<type-error>#.


Exported from:
------------------------------
- #Module common-dylan#
- #Module Dylan#


Arguments:
------------------------------
value - An instance of #<object>#. 
type - An instance of #<type>#. 


Values:
------------------------------
value - An instance of #<object>#.
