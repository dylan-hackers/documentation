Variable: *print-pretty?*
==============================

Specifies whether pretty printing is on. If on, prints object indented with
proper line breaks, according to the formatting specifications given to
#pprint-logical-block#. Users should have little use for this function. See
#Module pprint# for details.


Exported from:
------------------------------
- #Module print#


Type:
------------------------------
#<boolean>#


Value:
------------------------------
unknown
