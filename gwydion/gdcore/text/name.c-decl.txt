Macro: c-decl
==============================

Create a global C declaration. If used inside a Dylan function, the
declaration will be created after the Dylan function in the compiler output.


Exported from:
------------------------------
- #Module system#


Macro call:
------------------------------
: c-decl ( (decl) )


Arguments:
------------------------------
decl - A literal instance of #<string>#. A global C declaration.
