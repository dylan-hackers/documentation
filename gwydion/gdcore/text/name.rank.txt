Generic: rank
==============================

Returns the number of dimensions of an array. The method for #<array>#
computes _rank_ by calling #size# on the dimensions of _array_.

Exported from:
------------------------------
- #Module common-dylan#
- #Module Dylan#


Modifiers:
------------------------------
open


Arguments:
------------------------------
array - An instance of #<array>#. 


Values:
------------------------------
rank - An instance of #<integer>#.
