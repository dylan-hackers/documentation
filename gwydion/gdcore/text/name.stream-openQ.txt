Generic: stream-open?
==============================

Checks to see if a stream is open. If there is content available for the
stream, then it is open.


Exported from:
------------------------------
- #Module streams-protocol#
- #Module streams#


Modifiers:
------------------------------
open


Arguments:
------------------------------
stream - An instance of #<stream>#. 


Values:
------------------------------
open? - An instance of #<boolean>#.
