Harlequin Dylan Documentation

Documents

Welcome to the Harlequin Dylan on-line documentation set. It contains the following documents:
  • Getting Started with Harlequin Dylan, which describes Harlequin Dylan's interactive development environment.
  • Building Applications Using DUIM, which describes how to use DUIM (Dylan User Interface Manager), Harlequin Dylan's portable window programming toolkit.
  • The Common Dylan and Harlequin Extensions library reference, which describes Harlequin's implementation of the Dylan language, a common set of Dylan libraries, and a library interchange mechanism. The common libraries provide many language extensions, a threads interface, and object finalization.
  • The System and I/O library reference, which describes Harlequin Dylan's printing and output formatting modules, a streams module, a sockets module, and modules providing an interface to operating system features such as the file system, time and date information, and the host machine environment.
  • The DUIM library reference, which describes the libraries forming DUIM (Dylan User Interface Manager), Harlequin Dylan's portable window programming toolkit. It complements Building Applications Using DUIM.
  • The C FFI and Win32 library reference, which describes a foreign function interface library for interoperation with C programs, and several libraries providing a low-level interface to the Microsoft Win32 API.
  • The OLE, COM, ActiveX and DBMS library reference, which describes high and low level interfaces to COM, OLE, and ActiveX component technology, and generic DBMS support, through SQL with an ODBC backend.
  • The Developing Component Software with CORBA, a tutorial and reference for CORBA interoperability using the Harlequin Dylan ORB.
  • The Dylan Reference Manual by Andrew Shalit, which is the definitive reference for the Dylan language.
  • Dylan Programming by Feinberg, Keene, Mathews,and Withington, which is a programming tutorial with progressive examples in Dylan.

Printable versions of these documents

Under the Documentation folder in your Harlequin Dylan installation, you can find versions of these documents suitable for printing. These documents are also accessible from the Harlequin Dylan group in the Start menu. The document files are in Portable Document Format (PDF). You will need a copy of Adobe's Acrobat Reader application to read these files.
